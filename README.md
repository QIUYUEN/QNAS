# QNAS v13

本版基于 v12 继续优化：

- 左侧用户信息去掉白色/浅色底色。
- 用户角色文字移动到用户名后面显示，例如：`admin 管理员`。
- 左侧磁盘列表默认无边框，鼠标移上去才显示边框和背景。
- 增加自动端口能力：默认请求 8088，如被占用，会自动尝试 8089、8090……直到 `server.portRangeEnd`。

端口配置在 `config.json`：

```json
"server": {
  "host": "0.0.0.0",
  "port": 8088,
  "autoPort": true,
  "portRangeEnd": 8188,
  "jwtSecret": "please-change-this-random-secret"
}
```

如果自动切换端口，启动窗口会显示新的本机/局域网访问地址。
